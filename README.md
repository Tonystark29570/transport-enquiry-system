# 🚌 Transport Enquiry System

## 📖 Overview

The **Transport Enquiry System** is a Java-based desktop application developed using JavaFX and Maven as part of an academic project.
The system is designed to provide users with a simple and efficient way to access transport-related information such as routes and basic details.

---

## 🎯 Objective

* To develop a user-friendly transport enquiry system
* To simplify access to transport information
* To apply Java and JavaFX concepts in a real-world application
* To build a structured application using Maven

---

## ⚙️ Features

* 🔍 Search and view transport details
* 🖥️ Interactive graphical user interface (JavaFX)
* ⚡ Fast and simple data handling
* 📦 Structured project using Maven

---

## 🛠️ Technologies Used

* **Programming Language:** Java
* **Framework:** JavaFX
* **Build Tool:** Maven
* **IDE:** Apache NetBeans / IntelliJ IDEA

---

## 🏗️ Project Structure

```
transport-enquiry-system/
│── src/main/java/        # Source code
│── src/main/resources/   # UI files (if any)
│── pom.xml               # Maven configuration
```

---

## ▶️ How to Run the Project

### Prerequisites

* Java 11 or higher
* Maven installed

### Steps

1. Open terminal in the project folder
2. Run the command:
   mvn clean javafx:run

---

## 📌 Academic Context

This project was developed for academic purposes as part of the B.Tech Information Technology curriculum.
It demonstrates the application of object-oriented programming, GUI development, and build automation using Maven.

---
